# Weather App
A simple Weather App that shows the current weather for any city.

## Features
- Search weather by city name
- Shows temperature, humidity, and weather description
- Responsive design

## How to Use
1. Open `index.html` in your browser.
2. Enter a city name and press Enter or click Search.
3. See the weather instantly.
